# Welcome to my personal portfolio! 
The goal of this site is to present my work, skills, and background in a clean and modern interface inspired by a web browser and folder theme.

## Tech stack
- Next.js — App Router for routing, server components, and optimized performance
- React — UI rendering
- Tailwind CSS — Utility-first styling
- Vercel — Deployment 

## Features 
- Responsive design on mobile and laptops
- Interactive animations and transitions
